# Third Party Code Registry

## Current status

- 当前没有从 P01—P19 复制源码到本仓库。
- 当前 P0 可靠性地基和产品代码均为 clean-room 独立实现。
- Python 与 npm 包属于声明式依赖，不代表采用其内部源码为本项目模块。
- 未确认来源、固定版本和许可证前，不复制外部源码、Prompt 包或数据制品。

## Required fields for future adoption

任何后续引入都必须记录：

1. 外部仓库或发行包的准确地址；
2. 固定 Commit、Tag 或精确版本；
3. 许可证与兼容性判断；
4. 实际采用的文件、函数或数据；
5. 本项目所做修改；
6. Copyright / NOTICE / attribution 要求；
7. 对应 GitHub Issue、PR 与验证方式。

研究档案中的“采用某机制”默认表示重新实现机制，不表示复制原项目代码。
