# 统一基线前的详细文档归档

本目录保存统一产品与系统设计基线 V1.0 建立前仍有追溯价值的详细文档。它们不再分别决定当前产品方向，但没有因“合并成统一基线”而被删减。

需要当前产品目标、范围、进度和开发顺序时，先看 `docs/CURRENT_BASELINE.md`。需要下列详细内容时，再回查对应原文：

- `PRODUCT_BASELINE.md`：统一前的产品目标、用户流程、页面和阶段范围。
- `ROADMAP.md`：统一前的 P0～P6 开发路线和当时进度。
- `IMPLEMENTATION_AUDIT.md`：各区域哪些是正式地基、首条闭环、需升级或未开发。
- `ISSUE_BACKLOG.md`：M0 工程任务、验收条件和未完成事项的详细清单。
- `RESEARCH_TRACEABILITY.md`：P01～P19 与工程模块、机制、取舍和验证方式的对应关系。

归档规则：归档表示“不再作为唯一当前判断入口”，不表示内容无用、可以删除或可以只保留摘要。当前基线出现概括不够时，应回查这些原文；其中仍有效的判断应补回当前基线或对应工程专项文档。
